❌ COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════════════

Issue: "Stored Procedure not found"
  → Check SQL_DEPLOYMENT_GUIDE.sql was executed
  → Verify procedure names in Object Explorer
  → Check database name and server connection

Issue: "Cannot delete class"
  → Class has students enrolled
  → Solution: Remove students first or
  → Move students to another class

Issue: "Duplicate key error when adding"
  → Class code already exists
  → Solution: Use different class code

Issue: "Form doesn't appear"
  → Check menu event handler connected
  → Verify formMain.Designer.cs has Click binding
  → Run Build → Clean Solution → Rebuild

Issue: "Application crashes on search"
  → Database not connected
  → Solution: Check DBHelper configuration
  → Verify SQL Server is running
